# Game of Life

This is an implementation of [Conway's Game of Life](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life) that I made using [LWJGL](https://www.lwjgl.org/).

## How to Run

* Make sure you have Java 17+ installed
* Execute `./gradlew run`
